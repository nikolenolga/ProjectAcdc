: разделитель
? маркер начала вопроса
< ответ, ведет к поражению
> ответ, ведет к победе
= ответ, ведет к следующему вопросу

Вопрос:
? : номер вопроса : текст вопроса

Список ответов к вопросу
= : номер следующего вопроса : текст ответа : финальный текст ответа<br>
< : 0 : текст ответа : финальный текст ответа
> : 0 : текст ответа : финальный текст ответа
