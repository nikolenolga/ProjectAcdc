: разделитель <br>
? маркер начала вопроса <br>
< ответ, ведет к поражению <br>
> ответ, ведет к победе <br>
= ответ, ведет к следующему вопросу <br>
<br>
Квест: <br>
: name : Имя квеста <br>
: description : Описание квеста <br>
<br>
Вопрос: <br>
? : номер вопроса : текст вопроса <br>
<br>
Список ответов к вопросу <br>
= : номер следующего вопроса : текст ответа : финальный текст ответа <br>
< : 0 : текст ответа : финальный текст ответа <br>
> : 0 : текст ответа : финальный текст ответа <br>
